Налаштування гіта першій раз. Виконуються один раз після встановлення

git config --global user.name "YOUR NAME HERE"

git config --global user.email "YOUR EMAIL HERE"

Створення репозіторію або актівація гіта у папці

git init

Показати інформацію про репозіторій, в якому він зараз стані

git status

Додати файли до комміту ( можемо використовувати . для додавання усіх файлів)

git add

Зробити комміт

git commit -m "YOUT TEXT MESSAGE"

Додати новий віддалений репозиторій

git remote add ПСЕВДОНІМ АДРЕССА

Вигрузити зміни на віддалений репозиторій

git push ПСЕВДОНІМ_РЕПОЗИТОРІЮ НАЗВА_ГІЛКИ
